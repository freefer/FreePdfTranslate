# UI 设计说明

## 🎨 现代化扁平设计

本项目采用了**现代化扁平设计风格**，无需额外的第三方UI库，完全基于原生 Windows Forms + 自定义样式实现。

---

## ✨ 设计特点

### 1. 扁平化设计
- ✅ 无边框按钮（FlatStyle.Flat）
- ✅ 简洁的线条和布局
- ✅ 清晰的视觉层次
- ✅ 现代化的间距和留白

### 2. 颜色方案

**主色调 - Material Blue**
- 主要蓝色: `#2196F3` (RGB 33, 150, 243)
- 深蓝色悬停: `#1976D2` (RGB 25, 118, 210)

**辅助色**
- 橙色（翻译按钮）: `#FF5722` (RGB 255, 87, 34)
  - 悬停: `#E64A19` (RGB 230, 74, 25)
- 绿色（保存按钮）: `#4CAF50` (RGB 76, 175, 80)
  - 悬停: `#388E3C` (RGB 56, 142, 60)
- 白色: `#FFFFFF`
- 浅灰背景: `#F5F5F5` (RGB 245, 245, 245)
- 深灰文本: `#424242` (RGB 66, 66, 66)
- 中灰文本: `#646464` (RGB 100, 100, 100)

### 3. 字体设计

- **主字体**: Segoe UI（Windows 现代字体）
- **标题**: 18px Bold（加粗）
- **副标题**: 14px Bold
- **按钮**: 10-11px Bold
- **正文**: 10px Regular
- **状态栏**: 10px Regular

### 4. Emoji 图标

使用 Emoji 图标代替传统图标，更加现代和直观：

- 📁 选择 PDF - 文件选择
- 📄 原始文档 - 左侧面板标题
- ✨ 翻译结果 - 右侧面板标题
- ◀ ▶ 上一页/下一页 - 导航
- 🚀 开始翻译 - 翻译按钮
- ⏳ 翻译中 - 翻译进行中
- 💾 保存 PDF - 保存按钮
- ✓ 成功提示 - 状态信息
- 🌐 全局图标 - 标题栏

## 💡 交互效果

### 按钮悬停效果

1. **蓝色按钮** (导航、选择)
   - 正常: #2196F3
   - 悬停: #1976D2 (变深)
   - 光标: Hand (手型)

2. **白色按钮** (顶部选择)
   - 正常: 白色
   - 悬停: #F5F5F5 (浅灰)
   - 光标: Hand

3. **橙色按钮** (翻译)
   - 正常: #FF5722
   - 悬停: #E64A19 (变深)
   - 光标: Hand

4. **绿色按钮** (保存)
   - 正常: #4CAF50
   - 悬停: #388E3C (变深)
   - 光标: Hand

### 状态变化

**翻译过程中**:
- 翻译按钮文字: "🚀 开始翻译" → "⏳ 翻译中..."
- 翻译按钮: 禁用状态
- 选择PDF按钮: 禁用状态
- 进度条: 实时更新

**翻译完成后**:
- 保存按钮: 启用（从灰色变为绿色）
- 翻译按钮: 恢复 "🚀 开始翻译"

## 🎯 设计原则

1. **简洁至上**: 去除不必要的装饰，专注功能
2. **一致性**: 统一的颜色、字体、间距
3. **可访问性**: 清晰的对比度，易读的文字
4. **反馈明确**: 每个操作都有清晰的视觉反馈
5. **美观实用**: 既美观又不影响功能性
